# Jupyter Lab Keyboard Shortcuts (macOS)

A comprehensive quick-reference guide for Jupyter Lab keyboard shortcuts to optimize your development workflow.

## File Operations & Saving

| Command | Keybinding |
| :--- | :--- |
| Save Notebook | <kbd>Cmd</kbd> + <kbd>S</kbd> |
| Save Notebook As… | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>S</kbd> |
| New Launcher | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>L</kbd> |
| Print… | <kbd>Cmd</kbd> + <kbd>P</kbd> |
| Close Tab | <kbd>Alt</kbd> + <kbd>W</kbd> |
| Reopen Closed Document | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>T</kbd> |
| Close and Shut Down Notebook… | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Q</kbd> |

## Cell Execution & Editing

| Command | Keybinding |
| :--- | :--- |
| Run Selected Cell | <kbd>Shift</kbd> + <kbd>Enter</kbd> |
| Undo | <kbd>Cmd</kbd> + <kbd>Z</kbd> |
| Redo | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>Z</kbd> |

## Search & Replace

| Command | Keybinding |
| :--- | :--- |
| Find… | <kbd>Cmd</kbd> + <kbd>F</kbd> |
| Find Next | <kbd>Cmd</kbd> + <kbd>G</kbd> |
| Find Previous | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>G</kbd> |
| Find and Replace… | <kbd>Cmd</kbd> + <kbd>H</kbd> |

## Navigation & Tab Management

| Command | Keybinding |
| :--- | :--- |
| Activate Next Tab | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>]</kbd> |
| Activate Previous Tab | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>[</kbd> |
| Activate Next Tab Bar | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>.</kbd> |
| Activate Previous Tab Bar | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>,</kbd> |
| Activate Previously Used Tab | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>'</kbd> |
| Search Tabs and Running Sessions | <kbd>Alt</kbd> + <kbd>Cmd</kbd> + <kbd>A</kbd> |

## View & Sidebar Layout

| Command | Keybinding |
| :--- | :--- |
| Fullscreen Mode | <kbd>F11</kbd> |
| Simple Interface | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>D</kbd> |
| Show Left Sidebar | <kbd>Cmd</kbd> + <kbd>B</kbd> |
| Show Right Sidebar | <kbd>Cmd</kbd> + <kbd>J</kbd> |

## Tools, Panels & Settings

| Command | Keybinding |
| :--- | :--- |
| Activate Command Palette | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>C</kbd> |
| Show Keyboard Shortcuts… | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>H</kbd> |
| File Browser | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>F</kbd> |
| Table of Contents | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>K</kbd> |
| Extension Manager | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>X</kbd> |
| Sessions and Tabs | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>B</kbd> |
| Property Inspector | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>U</kbd> |
| Debugger Panel | <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>E</kbd> |
| Pause (Debugger) | <kbd>F9</kbd> |
| Show Contextual Help | <kbd>Cmd</kbd> + <kbd>I</kbd> |
| Settings Editor | <kbd>Cmd</kbd> + <kbd>,</kbd> |

---
*Tip: You can always access and modify these shortcuts by pressing <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>H</kbd> to open the built-in Keyboard Shortcuts Editor inside Jupyter Lab.*
